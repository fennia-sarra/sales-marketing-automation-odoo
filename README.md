# Sales & Marketing Automation using Odoo ERP

## Project Overview

This project focuses on automating sales and marketing processes using Odoo ERP.

The system automates:
- Lead generation
- CRM opportunity management
- Quotation generation
- Customer follow-up
- Invoice generation
- Marketing workflows
- Project task creation

---

## Objectives

- Improve sales efficiency
- Reduce manual tasks
- Automate marketing operations
- Enhance customer relationship management
- Monitor business KPIs
